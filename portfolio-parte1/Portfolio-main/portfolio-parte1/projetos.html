Projeto: Capacitor Caseiro

Descrição: O projeto consiste em demonstrar a possibilidade de fazer um power bank caseiro,
utilizando recursos domiciliares, podendo armazenar qualquer tipo de energia.

Materiais

  1. Um recipiente plástico
  2. Fios de cobre
  3. Rolo de alumínio
  4. Água com sal
  5. Fita isolante
  6. Uma haste de cobre
